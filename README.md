# descargaPowerBI
con este script de python se puede descargar un informe de power bi, para ello se emplea la autenticación por aplicación, se descarga la info en un html temporal, se pasa cada pagina del reporte capturando una imagen a traves de la visualizacion en selenium y se unen las imagenes en un pdf. No se ha logrado la interaccion con los filtros de power bi
